TP1 : Installation et configuration de GIT

Le but de ce TP est de découvrir GIT, l'installer, et le configurer

Téléchargement de l'installeur et configuration de votre environnement de travail

Récupérer l'installeur GIT pour votre système d'exploitation

https://git-scm.com/

https://gitforwindows.org/

Télécharger et installer votre IDE préféré (sublime text, VSCode, brackets, notepad++)

Installation

Lancer l'installation et suivre le guide

Pour windows : si demandé, penser à cocher la case "ajouter git.exe dans la variable d'environnement %PATH%"

Ensuite :

  • Lancer une invite de commandes, GitBash sur windows ou un shell classique sur linux/osx
  • Vérifier que git est bien disponible en ligne de commande en tapant simplement git -v

Configuration

Avec la commande git config, configurer de manière globale:

  • votre identité (nom, email)
  • votre éditeur de texte préféré
  • (...)

Vérifier que la configuration est effective

$ git config --list

Découverte des commandes disponibles

Lister les commandes git et repérer les similitudes avec les autres CVS que vous connaissez

  • Avec la commande git help, lister toutes les commandes disponibles.
  • Pour chaque commande disponible, utiliser la commande git <command> -- help

Création d'un premier dépôt et découverte des commandes de base

Utiliser la commande git init pour créer votre premier dépôt.

$ mkdir mon-projet

$ cd mon-projet

$ git init

Faire une modification puis un commit

$ echo "# Mon projet" > README.md

$ git status

$ git add README.md

$ git commit –m "version initiale du projet"

$ git status

$ git log

Faire ensuite une autre modification et un deuxième commit
